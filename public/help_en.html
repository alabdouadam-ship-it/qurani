Help

Welcome to Qurani. This guide summarizes the main features and how to use them on mobile and web.

Home and Settings

  • Greeting: The home screen greeting is localized. If you set your name in Preferences, it will appear.
  • Preferences: Set language, theme, Arabic font, and Quran font size. Font size applies across Read Quran, Repetition, and Search.
  • Web layout: On web, home and settings use a compact grid layout. Some mobile-only features are hidden (see below).

Read Quran

  • Navigation: Select a surah and read. Highlighted/bookmarked ayahs jump directly to the correct page and scroll into view.
  • Audio: When playing an ayah, the app prefers locally downloaded audio. If none and no internet, a clear message appears.
  • Font size: Matches your Preferences font size.

Listen Quran

  • Full Surah: Play full surahs. If downloaded, playback works offline.
  • Verse by verse: Prefers local files first; otherwise streams if online. Shows a clear message if offline with no local files.
  • Download buttons: Hidden if the full set is already downloaded.

Offline Audio

  • Categories: Full surahs and verse-by-verse for the selected reciter. Displays X/Y progress and reciter name.
  • Editions: You can also download Arabic–English, Arabic–French, and Tafsir (Muyassar) audio. Local files are used first where relevant.
  • Web: The Offline Audio option is hidden on web; downloads are mobile-only.

Repetition Memorization

  • Range playback: Choose a surah and range; audio prefers local files, then internet.
  • Versions: Edition names are localized.
  • Font size: Uses your Preferences font size.

Search

  • Mobile: Fast local FTS search.
  • Web: In-memory search over assets (no database).
  • Audio: Plays locally if available; otherwise streams with clear offline errors.

Memorization Stats

  • Aggregation: Shows all surahs and ajzaa (even untested) across two tabs, with test count and average percentage.
  • Sorting: Sorted by percentage, then Quran order.
  • No last/best: Uses aggregated results only (no last/best summaries).

Prayer Times

  • City: The app shows the city (when available) and supports testing Adhan notifications.
  • Monthly view: The second column displays the full date of the counterpart calendar (Hijri/Gregorian). Imsak is included.
  • PDF: Export mirrors the table, including Imsak. On web, the PDF export button is hidden.
  • Web: Qibla and Tasbeeh are hidden on web.

Connectivity and errors

  • Local-first: Audio playback always prefers downloaded files.
  • Offline errors: If no local file and no internet, a clear message explains that internet is required.

Privacy

  • Installation ID: The app creates a per-installation ID (UUID) to distinguish installs (not a hardware ID).
  • Device info: Non-identifying device info is collected once and stored locally; see the Privacy Policy for details.

Tips

  • If something doesn’t appear on web (e.g., Qibla, downloads), it is by design to keep the web build lightweight.
  • Use Preferences to adjust language, theme, font, and reciter.
