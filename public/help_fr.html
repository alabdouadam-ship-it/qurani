Aide

Bienvenue dans Qurani. Ce guide résume les fonctionnalités principales sur mobile et web.

Accueil et Réglages

  • Accueil : salutation localisée. Si vous définissez votre nom, il s’affichera.
  • Préférences : langue, thème, police arabe, taille de police. La taille s’applique à Lire, Répétition et Recherche.
  • Web : agencement en grille compact. Certaines fonctions mobiles sont masquées (voir ci-dessous).

Lire le Coran

  • Navigation : choisissez une sourate. Les versets marqués s’ouvrent directement sur la bonne page avec défilement.
  • Audio : préfère les fichiers locaux ; sinon diffusion en ligne. Message clair en hors-ligne sans fichiers.
  • Taille de police : suit les Préférences.

Écouter le Coran

  • Sourate complète : fonctionne hors-ligne si téléchargée.
  • Verset par verset : local d’abord, sinon en ligne ; message clair si hors-ligne sans fichiers.
  • Boutons de téléchargement : masqués si tout est déjà téléchargé.

Audio hors-ligne

  • Catégories : sourates complètes et verset par verset pour le récitateur sélectionné, avec progression X/Y.
  • Éditions : téléchargements disponibles pour Arabe–Anglais, Arabe–Français et Tafsir (Muyassar). Utilisation locale prioritaire.
  • Web : l’option Audio hors-ligne est masquée.

Mémorisation par répétition

  • Plage : choisissez une sourate et une plage ; audio local prioritaire.
  • Éditions : noms localisés.
  • Taille de police : suit vos Préférences.

Recherche

  • Mobile : recherche locale rapide (base de données).
  • Web : recherche en mémoire sur les assets.
  • Audio : local si disponible ; sinon diffusion, avec messages clairs en hors-ligne.

Statistiques de mémorisation

  • Aggregation : toutes les sourates et juz (même non testés) sur deux onglets, avec nombre de tests et moyenne.
  • Tri : par pourcentage puis ordre du Coran.
  • Sans dernier/meilleur : uniquement les résultats agrégés.

Heures de prière

  • Ville : affichage de la ville (si disponible) et test d’Adhan.
  • Mensuel : la 2ᵉ colonne affiche la date complète du calendrier opposé (Hégirien/Greg.). Imsak inclus.
  • PDF : l’export reflète le tableau, Imsak inclus. Sur le web, le bouton PDF est masqué.
  • Web : Qibla et Tasbih masqués.

Connectivité et erreurs

  • Local d’abord : lecture audio avec priorité aux fichiers téléchargés.
  • Hors-ligne : message explicite si aucun fichier local et pas d’internet.

Confidentialité

  • ID d’installation : UUID propre à l’installation (pas un ID matériel).
  • Infos appareil : collecte non identifiante, stockée localement ; voir la Politique de confidentialité.

Astuce

  • Si une fonction n’apparaît pas sur le web (ex. Qibla, téléchargement), c’est intentionnel pour alléger la version web.
  • Utilisez les Préférences pour régler la langue, le thème, la police et le récitateur.
