Help

This page summarizes how to use Qurani across platforms.

  • Read: navigate to surahs; bookmarks jump to the correct page.
  • Listen: full surah and verse-by-verse. Local files preferred; clear offline messages.
  • Offline audio: manage downloads (mobile). Web hides this option.
  • Repetition: play ranges with local-first audio.
  • Search: local DB on mobile; in-memory on web.
  • Stats: Surahs/Juz tabs show test count and averages; sorted by percentage.
  • Prayer times: monthly view includes Imsak; PDF export hidden on web.
  • Preferences: language, theme, Arabic font, font size, reciter.
  • Privacy: installationId (UUID), non-identifying device info; see Privacy Policy.
